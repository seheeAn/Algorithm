## Commit Rule
- Solve - 혼자 해결한 문제
- Hint - 블로그 등 다른 사람 풀이를 참고 해서 해결한 문제
- Fail - 시도했지만 해결하지 못한 문제
- Improve - 코드를 개선 및 수정한 문제
- Docs - 코드 외 문서 수정 및 추가

## Issue
- [Solve]: 나중에 풀어보고 싶은 문제가 생기면 Issue로 기록  
- [Improve]: 이미 풀었지만 다시 봐야할 문제 Issue로 기록
- [SQL]: SQL 쿼리 문제 모음
